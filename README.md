# 🚀 Smart Study AI

An AI-powered learning assistant built using Flask and Offline AI (Ollama).

## 🔹 Features

* 🤖 AI-based question answering (math, coding, theory)
* 📊 Performance tracking dashboard
* 📺 YouTube learning recommendations
* ⚡ Clean chat interface
* 💻 Works with offline AI models (no cloud dependency)

## 🧠 Tech Stack

* Python (Flask)
* SQLite
* Ollama (Local LLM)
* HTML, CSS, JavaScript

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/Smart-Study-AI.git
cd Smart-Study-AI
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run Ollama (for AI):

```
ollama run phi
```

4. Start the app:

```
python app.py
```

## 📌 Note

This project uses a local AI model, so it runs completely offline.
